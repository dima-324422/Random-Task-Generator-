# Random Task Generator

**Автор:** Бондарев Дмитрий

**Описание:** Простое приложение для генерации случайных задач с фильтрацией по категориям, сохранением истории и загрузкой из файла.

**Примеры использования:**
- Запустить приложение: `python main.py`
- Сгенерировать задачу — нажать кнопку.
- Добавить новую задачу — ввести текст и выбрать категорию.
- Сохранить/загрузить историю — использовать соответствующие кнопки.
- Фильтровать задачи по типу — выбрать категорию в выпадающем списке.
